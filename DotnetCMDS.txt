run and build OrderPlus
dotnet run --project ./OrderPlus/OrderPlus.csproj

Build
dotnet build

publish exe
dotnet publish OrderPlus -c Release 
or
dotnet publish OrderPlus/OrderPlus.csproj -c Release



create Solution
dotnet new sln -n POSPlus

create Project
dotnet new classlib -o Common

add proj to Solution
dotnet sln add Common/Common.csproj

reference one proj to another
dotnet add OrderPls/OrderPls.csproj reference Common/Common.csproj

add db to env variable
[Environment]::SetEnvironmentVariable("OrderPlus_DB", "Server=localhost\SQLEXPRESS;Database=POSPlus;Trusted_Connection=True;TrustServerCertificate=True;", "User")
